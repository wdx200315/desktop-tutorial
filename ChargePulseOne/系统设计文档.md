### 1.2 技术栈

| 层级 | 技术选型 | 说明 |
|-----|---------|------|
| 车主端 | Qt 6 + QML | 跨平台移动应用 |
| 管理端 | Qt 6 + Widgets | 桌面管理应用 |
| 服务端 | C++17 + STL | 高性能TCP服务器 |
| 数据库 | MySQL 8.0 | 关系型数据存储 |
| 通信协议 | JSON + TCP | 轻量级数据交互 |
| 实时推送 | WebSocket | 长连接推送 |

### 7.2 安全措施

| 安全措施 | 实现方式 | 说明 |
|---------|---------|------|
| 密码加密 | bcrypt | 防彩虹表攻击 |
| 传输加密 | TLS/SSL | HTTPS/WSS |
| Token认证 | JWT | 短期有效，支持刷新 |
| SQL注入防护 | 参数化查询 | MySQL预处理 |
| 请求限流 | Token Bucket | 防暴力破解 |
| 敏感数据 | AES加密 | 数据库敏感字段 |
