## 系统简介

### 系统概述

智序医院门诊智慧调度系统（SmartSched-HIS）是一套现代化的门诊排队管理解决方案，旨在优化患者就医流程、减少等待时间、提升医院运营效率。

### 核心功能

- **智能排队**：自动分配排队号，实时显示等待人数
- **多点叫号**：支持诊室门口屏、候诊区大屏同步显示
- **数据分析**：提供日报、月报、趋势分析等统计报表
- **B超调度**：专门的B超室排队管理，支持3台设备并行
- **安全加密**：全程AES-256加密传输，确保数据安全

## 快速开始

### 1. 服务端启动

```bash
# Linux
./bin/smartsched-server --config config/server.json

# Windows
smartsched-server.exe --config config\server.json
```

### 2. 客户端启动

**患者终端：**
```bash
./bin/smartsched-patient
```

**医生工作站：**
```bash
./bin/smartsched-doctor
```

**管理后台：**
```bash
./bin/smartsched-admin
```

### 操作步骤

#### 1. 选择科室
- 点击屏幕上方的科室按钮
- 系统会显示该科室当前等待人数和预计等待时间

#### 2. 输入患者信息
- 输入患者卡号或刷身份证
- 确认患者姓名

#### 3. 获取排队号
- 系统自动分配排队号
- 显示排队位置和预计等待时间

#### 4. 候诊
- 查看候诊区大屏
- 听到叫号后前往对应诊室

#### 5. 查询排队状态
- 可随时在终端查询当前排队情况
- 支持扫描回执单二维码查询

### 注意事项

- 请在工作时间（8:00-17:00）内挂号
- B超检查需空腹6小时以上
- 如需退号，请在叫号前操作

### 操作说明

#### 叫号
1. 点击「叫号」按钮
2. 系统通过扬声器和屏幕显示叫号信息
3. 患者到达后点击「完成诊疗」

#### 完成诊疗
1. 输入诊断结果
2. 填写处方信息（如有）
3. 添加医嘱
4. 点击「完成诊疗」

#### 转诊
1. 点击「转诊」按钮
2. 选择目标科室
3. 填写转诊原因
4. 确认转诊

#### 跳过
- 对于未到的患者，可点击「跳过」
- 系统自动叫下一位

## 常见问题

### Q1: 无法连接服务器？
**解决步骤：**
1. 检查网络连接
2. 确认服务端是否运行
3. 检查防火墙设置（端口8888）
4. 查看日志文件 `logs/server.log`

### Q2: 挂号失败？
**可能原因：**
- 科室已满号
- 系统维护中
- 信息填写错误

**解决方法：** 请咨询服务台工作人员

### Q3: 叫号无声音？
1. 检查音响设备连接
2. 确认音量设置
3. 检查系统声音配置

### Q4: 如何重置密码？
联系系统管理员，通过管理后台重置密码。

### Q5: 数据导出失败？
1. 检查存储空间
2. 确认导出格式
3. 减小导出时间范围
